# Service Image-Generation Prompts — The Walls

Image-generation prompts for Nano Banana — **4 distinct prompts per service** (one per
photo slot) for each of the 6 services on `pages/services.html` (24 prompts total). Each
service section on the page has one large main feature image plus three gallery images;
within each service the four shots are designed to complement rather than repeat each other:

- **Slot 1 — Hero / wide:** the finished work in its full setting
- **Slot 2 — Detail:** a tight close-up of materials and craftsmanship
- **Slot 3 — Context / in-progress:** the work being made, installed, or in use
- **Slot 4 — Main feature image:** a signature editorial showpiece for the large hero slot

**Shared specs for cohesion:** All images are **4:3 aspect ratio** (matching the service
image slots on the page). Every prompt uses the same visual language — photorealistic
commercial photography, an obsidian-black / warm-walnut / muted antique-gold (`#C9A96E`)
palette, warm cinematic directional lighting, full-frame camera — so the whole set looks
like one consistent shoot. No text, logos, or watermarks in any image.
